# Terminology Baseline

This file defines the canonical wording for user-facing UI copy.

## Core terms

- `Note` / `Notes` -> `随笔`
- `Post` / `Posts` -> `文章`
- `Photo` / `Photos` -> `照片`
- `Film photos` -> `胶片照片`

## Actions

- `Access key` -> `访问密钥`
- `Alt text` -> `Alt 文本`
- `Caption` -> `图注`
- `Publish now` -> `立即发布`
- `Clear draft` -> `清空草稿`
- `Manual publishing steps` -> `手动发布步骤`

## Consistency rules

- Prefer sentence case in English UI labels.
- Reuse the same wording for repeated actions across write pages.
- Keep Chinese functional labels direct and concise; avoid synonym drift across pages.
